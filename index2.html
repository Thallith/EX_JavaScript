Exercícios

Veja os exercícios

Clique no nome para abrir

Pares ou ímpares

Desenvolver um programa (Web) para exibir todos os números pares de um intervalo de 1 a 5000

Conversão

Desenvolva uma página que leia uma temperatura e converta a ela para os graus Fahrenheit ou Celsius.

Valor da venda

Desenvolva uma página leia o nome, o valor e a quantidade vendida de três produtos. Calcule o valor total da venda.

Gasto de um veículo

Desenvolva uma página que leia a quilometragem percorrida por um determinado veículo e o valor do combustível. Calcule o gasto dele, sabendo que o veículo percorre 8km/litro.

Custo para assentar piso

Desenvolva uma página para calcular o custo para assentar piso, sabendo que o preço é de 36,00 por m². O programa deve ler o comprimento e a largura total do cômodo.

Peso ideal

Tendo como dados de entrada a altura e o sexo de uma pessoa a partir de uma caixa de combinação, construa uma página que calcule seu peso ideal, utilizando as seguintes fórmulas: Para homens: (72.7*h) - 58 Para mulheres: (62.1*h) - 44.7 (h = altura)


Progressão aritmética

Crie uma página para calcular a P.A. (Progressão Aritmética). Receba do usuário o valor de início, o valor final e a razão da P.A. Crie dois botões para exibir o intervalo crescente e o decrescente.
